# CmdBlockBuster  
Tetris that supports most popular [Super Rotation System](https://tetris.fandom.com/wiki/SRS)  

Features:  
 - Hard drops
 - SRS rotaion
 - Naive gravity

 # Field
 It have 2 invisible top rows to spawn tetrominoes, it is enough if using SRS system
